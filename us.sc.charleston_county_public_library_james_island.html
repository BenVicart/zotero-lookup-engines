United States - South Carolina - Charleston County Public Library - James Island

This is a Zotero library lookup engine for Charleston County Public Library - James Island.

Unfortunately, Zotero has removed the ability to add these lookup engines to Zotero from a user interface (see https://github.com/zotero/zotero-connectors/issues/81). You can add the following snippet to the file named locate/engines.json in your Zotero data directory. To find out where your Zotero data directory is, check your Zotero preferences, Advanced tab for the data directory location. 

{
    "_name": "Charleston County Public Library - James Island",
    "_alias": "Charleston County Public Library - James Island",
    "_urlTemplate": "http://192.152.249.232/uhtbin/cgisirsi/x/0/0/5?searchdata1={rft:isbn}&srchfield1=GENERAL^SUBJECT^GENERAL^^ISBN&library=JAMES_IS",
    "_description": "Search Charleston County Public Library - James Island",
    "_urlParams": [],
    "_hidden": false,"_urlNamespaces": {
	"rft": "info:ofi/fmt:kev:mtx:journal",
	"z": "http://www.zotero.org/namespaces/openSearch#",
	"": "http://a9.com/-/spec/opensearch/1.1/"
    }
}

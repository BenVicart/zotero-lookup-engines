Italy - OPAC SBN Catalogo del Servizio Bibliotecario Nazionale

This is a Zotero library lookup engine for OPAC SBN Catalogo del Servizio Bibliotecario Nazionale.

Unfortunately, Zotero has removed the ability to add these lookup engines to Zotero from a user interface (see https://github.com/zotero/zotero-connectors/issues/81). You can add the following snippet to the file named locate/engines.json in your Zotero data directory. To find out where your Zotero data directory is, check your Zotero preferences, Advanced tab for the data directory location. 

{
    "_name": "OPAC SBN Catalogo del Servizio Bibliotecario Nazionale",
    "_alias": "OPAC SBN Catalogo del Servizio Bibliotecario Nazionale",
    "_urlTemplate": "https://nuovo-opac.sbn.it/risultati-ricerca-avanzata?fieldstruct%5B1%5D=ricerca.parole_tutte%3A4%3D6&struct%3A1001=ricerca.parole_almeno_una%3A%40or%40&fieldvalue%5B1%5D={z:title}&fieldaccess%5B1%5D=Any%3A1016%3Anocheck",
    "_description": "OPAC SBN",
    "_urlParams": [],
    "_hidden": false,"_urlNamespaces": {
	"rft": "info:ofi/fmt:kev:mtx:journal",
	"z": "http://www.zotero.org/namespaces/openSearch#",
	"": "http://a9.com/-/spec/opensearch/1.1/"
    }
}
